Project Overview


Title: Designing a new feature for U-M GPT.

Role & Responsibilities: UX Designer, in charge of creating task flows that effectively depict the functionality of the new focused feature I created for U-M GPT.

Tools: Figma.


Process & Approach


Context: Because U-M GPT is solely for University of Michigan students, my goal for this design was to add new features for U-M GPT that directly help ease the life of students on campus.


Design Solution: I decided to create a new feature that would allow students to create class folders within U-M GPT, and also allow students to upload documents (like class notes and syllabi) within that folder that U-M GPT can reference.
